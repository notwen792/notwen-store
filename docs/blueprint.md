# **App Name**: GridGazer

## Core Features:

- Profile Display: Display user profiles based on a grid layout.
- Neon Color Scheme: Implement the specified neon color scheme for a visually striking look.
- Font Implementation: Incorporate 'Inter' and 'Bebas Neue' fonts throughout the application.
- Interactive Cards: Cards will have a subtle glow effect upon hover to provide feedback to the user.

## Style Guidelines:

- Primary color: Magenta (#FF2D75) for main interactive elements, emphasizing the energetic theme.
- Background: Dark grey (#0A0A0A) to make the neon colors pop.
- Accent color: Cyan (#00F2FF), an analogous color to magenta on the color wheel, to provide contrast.
- Headline font: 'Bebas Neue', a sans-serif font, for titles and headings to create a modern and strong look.
- Body font: 'Inter', a sans-serif font, for body text and labels to ensure readability and a contemporary feel.
- Note: currently only Google Fonts are supported.
- Grid-based layout with rounded containers to achieve a modern, gaming-inspired interface.
- Subtle glow effects for hover states on interactive elements (buttons, cards).
